<h4 align="center">Installation Process</h4>

##### Clone repo:

```bash
git clone https://github.com/Azizbek98/RSSchool-NodeJS.git
```

##### Install and run:

```bash
cd [task_name]
npm install
npm start
```

Build project in production mode (if this mode exists):

```bash
npm run build
```
